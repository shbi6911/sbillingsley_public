import numpy as np
import matplotlib.pyplot as plt
def drawMoons(names, xpositions, xlim=[-500,500], labels=False, color=False):
    '''
    Draw a plot of the positions of moons relative to Jupiter.
    
    This function requires two input arguments. They should
    both be lists, and they should be the same size as each
    other. They are:
        
        moons -- a 1-dimensional list of moon names
        xpositions -- a 1-dimensional list of moon positions (in arcsec)
    
    For example:
        
        %matplotlib inline
        names = ['Io', 'Europa', 'Ganymede', 'Callisto']
        xpositions = [-20, 40, 80, -160]
        drawMoons(names, xpositions)
        
        (should display a plot of the moon positions)
    
    Options keyword arguments
    
        xlim = [-500,500]
            (set this to some other 2-element list to change
                the range over which the plot stretchs)
        labels = False
            (set this to True to provide text labels)
        color = False
            (set this to True to provide different colors for each moon)
            
    '''
    
    # since we're plotting only 1D positions, we make up y-values
    ypositions = np.zeros_like(xpositions)

    # we create a new figure, and set its size
    plt.figure(figsize=(10,.25))
    
    # if color output, set colors
    if color:
        colors=['orange','blue','magenta','green']
    else:
        colors=['black']*len(xpositions)
        
    # we plot the moons in their positions
    for i in range(len(xpositions)):
        plt.plot(xpositions[i], ypositions[i],
                    marker = '.',
                    linewidth=0,
                    color=colors[i])
    
    # if desired, we add text labels to all the moons
    if labels:
        for x, y, n in zip(xpositions, ypositions, names):
            plt.text(x, y+0.1, n, ha='center', va='bottom')
    
    # plot Jupiter in the center
    plt.plot(0,0, marker='o', markersize=10, color='black')    

    # set the x and y limits of the plot
    plt.xlim(*xlim)
    plt.ylim(-1,1)

    # turn off all axis labels (and the box around the plot)
    plt.axis('off')

    # make sure the plot shows to the screen
    plt.show()

Section 8: Calculate The Moons' Positions Over Time¶

In [22]:
times = range(50)    #create a list of even time steps from 0-49
positions = []       #create an empty list to store position data
for t in times:                          #this loop will calculate positions for moons at 1-day time steps
    positionsAtThisTime = []
    for i in range(len(moons)):          #this loop calculates positions for each of the moons
        positionsAtThisTime.append(orbitsizes[i]*sin(((2*pi*t)/periods[i]) + initialphases[i]))
    positions.append(positionsAtThisTime)               #store position data in a nested list
    drawMoons(moons, positionsAtThisTime, labels=True)  #this outputs a picture using drawMoons

Results as expected. Callisto, as the outermost moon, has a very wide orbit, with Io closest in. Callisto's orbital period was stored back at the beginning as just over sixteen days, and we see Callisto complete just over three full orbits in this 50-day period.

Section 9: Lists Within a List¶

  1. The length of the list positions is 50, with indexes 0-49.
  1. This length corresponds to the number of time points.
  1. The expression positions[0] is itself a list. It expresses the positions of each of the four Galilean moons at time point 0. Per the equation in the loop, these should be the values stores in initialPhases.
  1. The expression positions[1] is also a list. It expresses the positions of each of the four Galilean moons after one day has passed. It is calculated using the equation in the second nested loop.
